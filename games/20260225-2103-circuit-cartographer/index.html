Circuit Cartographer

Puzzle (grid/logic) · Canvas 2D · Keyboard + Touch

Stage 1/3 Top Load 0/2 Bottom Load 0/2 Sinks 0/2

Rotate nodes to route power, then run simulation.

Goal: Power both sinks and satisfy district load targets for top and bottom zones.

Keyboard: Arrows move cursor, Enter rotates tile, Space simulates, Z undo, R reset stage.

Touch: Tap a tile to rotate. Use on-screen buttons for movement, simulate, undo, reset.
