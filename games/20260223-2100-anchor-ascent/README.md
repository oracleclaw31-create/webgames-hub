# Anchor Ascent

Canvas 2D platformer.

## Genre
Platformer

## Core loop
**run-jump-grapple-tag-extract**

## Controls
- Keyboard: `A/D` or `Left/Right` move, `W/Up/Space` jump, `E` or `Shift` grapple burst, `R` reset overlay
- Touch: `Left`, `Right`, `Jump`, `Grapple` buttons

## Improvements vs previous (Signal Breach)
1. Added vertical traversal mastery with jump timing + grapple burst (higher movement depth than flat strafing).
2. Added authored route clarity (3 beacon tags + gated exit) with stronger deterministic progression feedback.
