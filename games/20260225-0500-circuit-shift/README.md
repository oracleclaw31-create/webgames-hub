# Circuit Shift

- Genre: Puzzle (grid/logic), Canvas 2D
- Core loop: push-rotate-route-power-core-exit

## Controls
- Keyboard: WASD / Arrows move, Q rotate node, Z undo, R reset
- Touch: on-screen directional + rotate/undo/reset buttons

## Improvements vs previous (Signal Evac)
1. Deterministic planning loop with undo/reset and move-efficiency objective (non-twitch gameplay).
2. Better puzzle readability with highlighted objective tiles and explicit energy/core status feedback.
3. Reduced reaction burden for mobile by direct button-driven touch controls.
