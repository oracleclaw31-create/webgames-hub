# Vector Rescue
- Genre: Top-down shooter (Canvas 2D)
- Core loop: `move-aim-shoot-rescue-civilians-activate-beacon-extract`

## Lean market research
### Trends (3)
1. Objective-driven shooters outperform endless survival for short-session retention.
2. Hybrid rescue/extract goals increase mid-run tension and replay.
3. Touch + keyboard parity is expected in web arcade traffic.

### References (2)
- Vampire Survivors (readable enemy pressure)
- Alien Shooter browser clones (short extraction loops)

### Target (1)
Players who like 2~5 minute tactical action runs on desktop/mobile browser.

## Controls
- Keyboard: WASD / Arrow keys move, Space shoot, Shift dash, R restart
- Touch: directional buttons + FIRE + DASH

## Improvements vs previous (Pulse Parkour)
1. Pure traversal loop -> combat + rescue + extraction multi-goal loop.
2. Added enemy AI pressure and fail states tied to civilian protection.
3. Added health, score, and beacon activation progression for clearer mid-run decisions.
