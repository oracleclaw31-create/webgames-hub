# Prism Surge
- Genre: Brick-breaker (Canvas 2D)
- Core loop: charge-launch-angle-control-break-shield-waves

## Lean market research
### Trends (3)
1. One-thumb + keyboard parity is now expected on browser arcade games.
2. Short roguelite-style wave escalation boosts replay retention.
3. Clear hit feedback and charge mechanics improve session depth.

### References (2)
- DX-Ball style angle control
- Shatterline brick shield layering patterns

### Target (1)
- Players wanting 3-6 minute skill-based arcade sessions on mobile/desktop.

## Improvements vs previous game (Switchyard Stacker)
1. Added real-time charge-shot + rebound mastery (from turn-based puzzle to action skill loop).
2. Added shield bricks (2HP) and wave phase escalation for better progression clarity.
