# Relay Grid

A no-build browser puzzle game where the player routes energy from Source to Goal by toggling relay nodes.

## Core Loop
1. Move through the grid.
2. Select/toggle relay tiles.
3. Create a valid power route and reach Goal before time runs out.

## Controls
- Keyboard: Arrow keys / WASD to move, Space to toggle nearby relay
- Touch: On-screen direction buttons + tap relay tile + Toggle button
