# Vector Siege

Fast top-down shooter built with Canvas2D.

## Core Loop
1. Move through the arena while enemies chase.
2. Fire to clear drones and build score.
3. Survive until time ends with enough eliminations.

## Controls
- Keyboard: `WASD` or Arrow keys move, `Space` fire
- Touch: D-pad + Fire

## Run
Open `games/20260223-1521-vector-siege/index.html` in a browser.
