# Reactor Rhythm

- Genre: Rhythm/timing (Canvas 2D)
- Core loop: read-cue-time-lane-hits-build-focus-maintain-stability

## Controls
- Keyboard: `A/S/K/L` hit lanes, `Space` activates Focus, `Enter` restarts after clear/fail.
- Touch: 4 lane buttons + `FOCUS` button.

## Lean market notes
- Trends: short-session rhythm loops, strong hit feedback, mobile input parity.
- References: Friday Night Funkin' timing readability, osu!mania lane precision cadence.
- Target: players wanting 3-6 minute skill runs with immediate retries.
