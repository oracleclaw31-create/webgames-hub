# Polarity Pendulum

- Genre: Physics action (Canvas 2D)
- Core loop: swing-flip-polarity-collide-shatter-maintain-control

## Controls
- Keyboard: Left/Right or A/D move torque, Space flip polarity, E pulse burst, R restart
- Touch: Left, Right, FLIP, PULSE buttons

## Goal
Use pendulum momentum and polarity switching to shatter unstable cores while preserving lives.
