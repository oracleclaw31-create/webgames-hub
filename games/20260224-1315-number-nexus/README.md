# Number Nexus

Canvas 2D **수학 퍼즐 게임**.

## Genre
- Puzzle (math)

## Core loop
- choose-number -> choose-operator -> hit-target-in-limited-moves

## Rules
- 매 라운드마다 목표값(Target)이 주어집니다.
- 숫자 카드와 연산(+,-,×,÷)을 조합해 현재값(Current)을 목표값과 같게 만드세요.
- 이동(연산) 기회는 3번입니다.

## Controls
- Mouse/Touch: 카드/연산/Apply/Next 버튼 클릭
- Keyboard: `1~4` 숫자카드, `Q/W/E/R` 연산(+,-,×,÷), `Enter` 적용, `Space` 다음 라운드

## Improvement vs previous (Orbit Breaker)
- 실시간 반사 액션에서 **결정형 수학 퍼즐 루프**로 전환
- 반응속도보다 연산 계획과 제한 턴 최적화 중심으로 난이도 설계
- 모바일에서도 동일하게 카드+연산 버튼 기반 조작 지원
