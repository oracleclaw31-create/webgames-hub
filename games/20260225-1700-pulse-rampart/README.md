# Pulse Rampart

Canvas 2D tower defense lite. Place slow/sniper turrets, trigger EMP pulse, and keep the core alive.

## Lean market research
- Trend 1: Short-session strategy loops with immediate readability perform well on web portals.
- Trend 2: Hybrid action+defense systems (active skills + passive towers) improve retention.
- Trend 3: Mobile-first touch parity is increasingly expected even for desktop-origin genres.
- Reference 1: *Kingdom Rush* style lane readability and wave cadence.
- Reference 2: *Isle of Arrows* style compact placement decisions and synergy focus.

## Controls
- Keyboard: `1/2` tower type, `Space` EMP pulse, `P` pause
- Touch/Mouse: tap/click to place selected tower
