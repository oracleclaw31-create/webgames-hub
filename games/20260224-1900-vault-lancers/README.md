# Vault Lancers
- Genre: Top-down shooter (Canvas 2D)
- Core loop: escort-drone-seal-rifts-kite-fire-extract

## Lean market research
### Trends (3)
1. Fast retry top-down arena runs are rising on web portals.
2. Dual-input parity (keyboard + touch) increases retention.
3. Objective-driven combat loops perform better than endless survival alone.

### References (2)
- Brotato-style arena readability and pacing
- 20 Minutes Till Dawn-style short-run intensity

### Target (1)
- Casual-core players wanting 5-10 minute tactical action sessions.

## Improvements vs previous (Relay Ruins)
1. Added ally-escort + objective sealing loop (not pure traversal).
2. Added pressure management via drone link meter + rescue boost skill.
3. Increased combat readability with dedicated touch FIRE/LINK actions.
