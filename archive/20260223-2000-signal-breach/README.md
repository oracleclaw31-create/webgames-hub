# Signal Breach

Canvas 2D top-down shooter.

## Genre
Top-down shooter

## Core loop
**strafe-mark-detonate-hack-extract**

## Controls
- Keyboard: WASD/Arrows move, Space fire, E hack/detonate, R retry
- Touch: directional + FIRE + HACK buttons

## Improvements vs previous (Orbit Forge)
1. Added mark-detonate combo layer (positioning + timing skill expression).
2. Added dual-objective extraction flow (relay hacking while handling drone pressure) with clearer mission completion state.
