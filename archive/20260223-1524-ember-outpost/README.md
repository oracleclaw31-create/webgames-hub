# Ember Outpost

Tower defense lite built with Canvas2D.

## Core Loop
1. Earn gold by defeating raiders.
2. Spend gold to place towers on build slots.
3. Survive escalating waves and protect base HP.

## Controls
- Keyboard: `B` toggles build mode
- Mouse/Touch: click/tap build slots to place towers

## Run
Open `games/20260223-1524-ember-outpost/index.html` in a browser.
