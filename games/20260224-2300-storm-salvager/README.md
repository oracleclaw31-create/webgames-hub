# Storm Salvager

- Genre: Survival arena (Canvas 2D)
- Core loop: move-sweep-emp-activate-beacons-survive

## Controls
- Keyboard: WASD/Arrows move, Space Sweep, Left Shift EMP, E Activate
- Touch: D-pad + Sweep/EMP/Activate buttons

## Objective
Activate all 3 beacons while surviving enemy swarms.
