# Neon Harvest

Fast arcade no-build web game.

## Core Loop
1. Collect neon shards in the arena.
2. Evade moving drones that reset your carry on hit.
3. Return to base to bank shards and reach 20 before time expires.

## Controls
- Keyboard: Arrow keys / WASD move, Space/Enter bank
- Touch: On-screen D-pad + BANK button

## Run
Open `games/20260223-1501-neon-harvest/index.html` in browser.
