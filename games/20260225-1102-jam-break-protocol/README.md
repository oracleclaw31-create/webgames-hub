# Jam Break Protocol
- Genre: Top-down shooter (Canvas 2D)
- Core loop: kite-shoot-clear-waves-manage-heat

## Controls
- Keyboard: WASD/Arrow move, Space shoot, Shift dash, R restart
- Touch: on-screen directional + shoot/dash buttons

## Improvements vs Previous (Skyhook Bastion)
1. Platform traversal 중심 루프에서 웨이브 전투 + 히트(heat) 관리 루프로 전환해 즉시성 강화
2. 단일 스테이지 퍼즐 성공조건 대신 wave escalation + 점수 시스템으로 재플레이성 강화
3. 자동 타깃 보정 + 모바일 터치 전투 버튼으로 조작 접근성 개선
