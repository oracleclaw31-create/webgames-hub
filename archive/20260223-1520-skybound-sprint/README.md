# Skybound Sprint

Short platformer built with Canvas2D.

## Core Loop
1. Run and jump across platforms.
2. Collect all 5 stars.
3. Reach the flag before the timer ends.

## Controls
- Keyboard: `A/D` or `Left/Right`, `W/Up/Space` jump
- Touch: Left, Right, Jump buttons

## Run
Open `games/20260223-1520-skybound-sprint/index.html` in a browser.
