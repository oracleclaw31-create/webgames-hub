# Iron Bulwark

- Genre: Tower defense lite (Canvas 2D)
- Core loop: place-upgrade-switch-target-priority-hold-waves

## Controls
- Keyboard: Arrow/WASD move cursor, Space build/upgrade, T switch tower type, P switch target priority, N start wave
- Touch: On-screen D-pad + Build/Upgrade + Switch Type + Target Mode + Start Wave buttons

## Objective
Defend the center lane from incoming enemies. Manage gold, choose tower type, and switch target priority to survive longer waves.
