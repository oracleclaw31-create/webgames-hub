# Turret Tide
- Genre: Tower defense lite, Canvas 2D
- Core loop: place-upgrade-defend-earn-scale

## Controls
- Keyboard: Arrow keys (cursor), B (build), U (upgrade), Space (start wave)
- Touch: Tap grid to select cell + on-screen buttons

## Improvements vs previous (Ember Brick)
1. Added strategic placement + upgrade economy instead of pure paddle reflex.
2. Added wave pacing with spawn management and defense planning depth.
3. Added grid cursor targeting parity for both keyboard and touch.
