# Tempo Forge

- Genre: Rhythm/timing (Canvas 2D)
- Core loop: hold-charge-release-on-beat-forge-chain
- Session: ~2-4 minutes

## Controls
- Keyboard: Hold `Space`, release on beat, `R` restart.
- Touch: Hold `HOLD`, then tap `RELEASE` on beat windows.

## Lean market research
