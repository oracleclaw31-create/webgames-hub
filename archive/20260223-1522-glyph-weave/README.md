# Glyph Weave

Classic sliding puzzle built with Canvas2D.

## Core Loop
1. Move the blank tile with arrow directions.
2. Reorder tiles to 1-8 sequence.
3. Minimize moves and solve quickly.

## Controls
- Keyboard: Arrow keys
- Touch: Up/Left/Right/Down buttons

## Run
Open `games/20260223-1522-glyph-weave/index.html` in a browser.
