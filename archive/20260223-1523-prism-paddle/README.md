# Prism Paddle

Brick-breaker built with Canvas2D.

## Core Loop
1. Move paddle to keep ball in play.
2. Break the full brick field.
3. Clear all bricks before losing all lives.

## Controls
- Keyboard: `Left/Right` or `A/D`
- Touch: Left/Right hold buttons

## Run
Open `games/20260223-1523-prism-paddle/index.html` in a browser.
