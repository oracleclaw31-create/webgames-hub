# Neon Quarantine
- Genre: Survival arena, Canvas 2D
- Core loop: dash-kite-charge-nodes-release-shockwave-survive

## Lean market research
### Trends (3)
1. Short-session survival loops with escalating pressure and clear 3-6 minute goals.
2. Single-stick + action-button mobile-friendly controls for cross-device play.
3. Distinct readability (high contrast telegraphs/HUD) favored in lightweight browser games.

### References (2)
- Brotato (survival arena pacing + compact runs)
- Vampire Survivors (pressure curves and readable enemy swarms)

### Target (1)
- Players who want a fast, replayable 3-5 minute action game that works on both desktop and phone.

## Controls
- Keyboard: WASD / Arrow keys move, Shift dash, Space pulse, R restart
- Touch: D-pad buttons move, Dash/Pulse buttons, Restart button

## Improvements vs previous (Turret Tide)
1. Switched from fixed-grid build flow to direct character control with dynamic kiting and dash timing.
2. Added node-charge objective with active pulse management for richer moment-to-moment decision making.
3. Improved mobile parity with explicit touch movement + action buttons instead of cell-tap interactions.
